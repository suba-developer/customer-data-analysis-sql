# Customer Data Analysis – SQL

## Project Overview

This project analyzes customer purchasing behavior using SQL to identify revenue opportunities and customer trends.

## Tools Used

- SQL
- Excel
- Power BI

## Dataset

Customer transaction dataset containing:
- Customer ID
- Product Category
- Purchase Amount
- Order Date
- Region

## Business Questions

1. Who are the top spending customers?
2. Which regions generate the highest revenue?
3. What are the monthly purchasing trends?
4. Which customer segments contribute the most revenue?

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOINS
- Subqueries
- Aggregate Functions

## Key Insights

- Identified top revenue generating customers.
- Analyzed customer purchasing trends.
- Generated revenue insight reports.
- Identified potential growth opportunities.
